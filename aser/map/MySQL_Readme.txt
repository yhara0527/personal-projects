Usage of this data is in accordance with the EPSG Dataset terms of use.  The terms of use can be found here: https://epsg.org/terms-of-use.html

To install this package, run the included scripts in the following order:
1) MySQL script to create the database tables (MySQL_Table_Script.sql)
2) MySQL script to insert the data (MySQL_Data_Script.sql)
3) MySQL script to create the database table constraints (MySQL_FKey_Script.sql)

Note that these scripts have been tested on MySQL 8.0.11 and use the UTF-8 encoding. If your database uses a different encoding you may encounter problems with international characters.
